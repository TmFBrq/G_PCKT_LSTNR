GEM_PCKT_LSTNR is a firmware for Cardputer Adv that scans for the various packets involved in WIFI communications and counts them. There are four screens involved.

Press "s" for the main dashboard screen

Press "w" for the waterfall screen

Press "m" for the map screen

Press "i" for the inspect screen

Press "b" to toggle the beeps on and off

Press "space" to toggle channel hopping on and off

Press the "Go" button on the top of the Cardputer to toggle the screen on and off
